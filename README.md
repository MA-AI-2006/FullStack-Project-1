Project 1: The Skin (Frontend)

Overview

A mobile-first, responsive landing page built strictly with raw HTML5, CSS3, and Vanilla JavaScript. No frameworks were used.

Key Features

1.Mobile-First Architecture: Optimized for small screens, scales up to desktop via CSS Grid.
2. 2025 Aesthetics: Implemented the Mocha Mousse color palette and clean typography.
3. Semantic HTML5: Proper use of <header>, <nav>, <main>, <aside>, and <footer>.
4. Vanilla JS Logic: Interactive hamburger menu and dynamic button states.
